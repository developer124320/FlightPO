Personalizzazione dei comandi

La personalizzazione dei comandi da tastiera (shortcuts) avviene attraverso il file qad_<lingua>_<regione>.pgp (utf-8).

<lingua> rappresenta il linguaggio corrente di QGIS (obbligatorio) e <regione> rappresenta la regione linguistica corrente (opzionale). Ad esempio qad_pt_br.pgp rappresenta il file in lingua portoghese della regione del brasile, qad_en.pgp è il file pgp per la lingua inglese. Il file è ricercato da QAD seguendo i percorsi indicati dalla variabile di sistema SUPPORTPATH.
